## Dynamic Price Number

```bash
fetch("https://widget.1price.co/api/v1/price_value", {
    method: "POST",
    headers: {
        "Content-Type": "application/json",
    },
    body: JSON.stringify({
        cookies: document.cookie,
    }),
    credentials: "include",
})
    .then((response) => response.json())
    .then((data) => {
        console.log("Products", data)
    })
    .catch((error) => {
        console.log(error)
    })
```
